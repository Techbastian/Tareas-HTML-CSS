Pseudo-Clases

Elemento:first-of-type

La pseudo-clase first-of-type de CSS representa el primer elemento de su tipo dentro de un grupo de elementos hermanos (o del mismo tipo).

.ejemplo1 a:first-of-type{
color: white;
font-family: Arial, Helvetica, sans-serif;
background-color: gray;
border-radius: 5px;
padding: 2px 0;
width: 30vw;
margin: 0 auto;
}

Elemento:focus

La pseudo-clase focus crea un foco en un elemento, que generalmente es la entrada de un formulario, y se activa al dar click o tocar el elemento.

.ejemplo2 .amarillo:focus{
color: blue;
background-color: yellow;
font-weight: 400;
}

Elemento:nth-child

La pseudo-clase nth-child permite seleccionar uno o dos elementos dependiendo de su posición (o posiciones) dentro de un grupo de hermanos.
odd selecciona los valores pares, como en este caso.
even selecciona los elementos impares.
An+B selecciona los elementos en funcion de un resultado.

#ejemplo3 p:nth-child(odd){
background-color: var(--fondodivisiones);
width: 40vw;
color: gray;
font-weight: 500;
}

Elemento:active

La pseudo-clase :active de CSS se activa al dar click sostenido sobre un elemento, comunmente se usa en enlaces < a> y en botones < input>. Indica que un elemento esta activado en su momento.

#ejemplo4 #boton1:active{
color:red;
font-weight: 600;
background: lightcyan;
}

Pseudo-Elementos

Elemento::after

::after nos permite crear un pseudo-elemento al final de un elemento seleccionado, por ejemplo nos permite añadir contenido para aclarar, o definir algun tipo de caracteristica. Este contenido se agrega con la pripiedad content.

#ejemplo6 p::after{
content: " -> Esta flechita va despues";
color: red;
}

Elemento::before

::before al igual que la etiqueta anterior, nos permite crear un pseudo elemento dentro de la caja del elemento seleccionado, pero esta vez es al principio de la caja, no al final.

#ejemplo7 p::before{
content: " ' ;) ' ";
color: green;
font-weight: 600;
}

Elemento::first-letter

::first-letter nos permite aplicarle estilos a la primera letra de un elemento bloque y que no sea precedido por ningún otro elemento.

#ejemplo8 p::first-letter{
color: red;
font-size: 120%;
font-weight: 500;
}

Elemento::first-line

::first-line es similar a la anterior, pero en este caso nos permite aplicar estilos a una línea completa de un elemento en bloque. La primera linea depende del ancho que se maneje dentro de la aplicación.

#ejemplo9 p::first-line{
background-color: white;
color: red;
font-size: 120%;
}

Elemento::marker

::marker lo que nos permite es modificar el puntero de una lista, aplicarle estilos o cambiarlo por completo.

#ejemplo10 li::marker{
content: "X ";
color: green;
font-weight: 600;
}
